How to run.

First you need the following:
	Nodejs
	express
	cors
	sqlite3

which can be done by:
	First install nodejs from this hyperlink https://nodejs.org/en.
	Then write the following commands in the terminal
	npm install express
	npm install cors
	npm install sqlite3


Then run the command 'node <path to Server.js>'
After running the command it should write the following into the terminal "Server is running on port 49293".
If the message does not appear please try running 'node <path to Server.js>' some time later.
